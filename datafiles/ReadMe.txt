A game starring Felix Sr. and Felix Jr. created in homage to Fix It Felix Jr. and Mario Bros. 

[Default Controls]
1 - 1 Player Start
2 - 2 Player Start
5 - Coin
6 - Coin
Left Arrow - P1 Left
Right Arrow - P1 Right
Ctrl - P1 Jump
D - P2 Left
G - P2 Right
A - P2 Jump
Esc - Quit
F12 Toggle Fullscreen
